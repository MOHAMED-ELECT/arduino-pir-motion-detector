# PIR Motion Detector (Arduino)

## 🚀 Description

This project is a motion detection system using a PIR sensor and an Arduino Uno.
It triggers a visual (LED) and sound (buzzer) alert when motion is detected.

## 🧰 Components

* Arduino Uno
* PIR Sensor (HC-SR501)
* LED
* Buzzer
* Resistors

## ⚙️ How it works

The PIR sensor detects motion and sends a signal to the Arduino.
If motion is detected, the LED and buzzer are activated.
